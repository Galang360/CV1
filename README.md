# Portfolio CV - Galang Febrian

Selamat datang di portfolio digital saya! Website ini menampilkan profil, pendidikan, pengalaman, keahlian, dan portofolio saya.

## 🎯 Tentang Proyek

Portfolio ini dibuat menggunakan **HTML, CSS, dan JavaScript** untuk menampilkan:
- **Profil Pribadi** - Informasi singkat tentang diri saya
- **Pendidikan** - Latar belakang pendidikan di Manajemen Informatika
- **Pengalaman** - Pengalaman kerja sebagai Teknisi Wifi RT/RW
- **Keahlian** - Tools dan bahasa pemrograman yang saya kuasai
- **Portofolio** - Proyek-proyek yang telah saya buat
- **Galeri** - Koleksi foto dan dokumentasi

## 🎮 Fitur

- ✨ Desain modern dan responsif
- 🎨 Animasi smooth dan interaktif
- 🔍 Lightbox untuk galeri foto
- 📱 Mobile-friendly design
- 🎯 Navigasi smooth scroll

## 🎲 Mini Game - QuickMath

Proyek ini juga menyertakan mini game "QuickMath" di folder `Mini_Game/`:
- Tantangan menjawab 10 soal matematika dalam 10 detik per soal
- Sistem scoring untuk mengetahui performa Anda
- Hasil akhir dengan detail jawaban benar/salah


## 🚀 Cara Menggunakan

1. Clone repository ini
```bash
git clone https://github.com/username/CV1.git
```

2. Buka file `index.html` di browser
```bash
open index.html
# atau double-click pada index.html
```

3. Nikmati portfolio dan coba game QuickMath!
